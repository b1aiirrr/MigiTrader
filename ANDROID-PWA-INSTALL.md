# MigiTrader PWA Installation Guide (Android)

## 📱 How to Install MigiTrader on Your Android Phone

MigiTrader is a **Progressive Web App (PWA)** - works like a native app without downloading from Google Play!

### ✅ What's Already Configured:
- **Standalone Mode** - Runs fullscreen like a native app
- **Offline Support** - Cached data works without internet
- **Android Icons** - Optimized 72px to 1024px icons with maskable variants
- **Quick Actions** - Long-press app icon for shortcuts
- **Portrait Lock** - Optimized for mobile trading
- **Theme Colors** - Metallic purple/blue theme

---

## 🚀 Installation Steps (Chrome/Edge):

### Method 1: Install Prompt (Easiest)
1. Open **Chrome** on your Android
2. Visit https://migi-trader.vercel.app/
3. Tap the **"Add to Home Screen"** banner at the bottom
4. Tap **"Install"** → Done! 🎉

### Method 2: Manual Install
1. Visit https://migi-trader.vercel.app/ in Chrome
2. Tap the **3-dot menu** (⋮) in the top-right
3. Select **"Add to Home Screen"** or **"Install app"**
4. Name it "MigiTrader" (or keep default)
5. Tap **"Add"**

### Method 3: Custom Icon (Advanced)
1. Long-press the home screen
2. Tap **"Widgets"**
3. Find **"Chrome"** → **"Website shortcut"**
4. Drag to home screen
5. Enter: https://migi-trader.vercel.app/

## 🛠 Troubleshooting:

### "Add to Home Screen" Not Showing?
- **Solution**: Make sure you're using **Chrome** (not Firefox/Samsung Internet)
- Visit the site 2-3 times to trigger the install prompt
- Check Chrome flags: `chrome://flags` → Enable "Add to Home Screen"

### App Won't Install?
- Clear Chrome cache: Settings → Apps → Chrome → Storage → Clear Cache
- Update Chrome to the latest version
- Ensure you have storage space (app is ~2MB)

### Offline Mode Not Working?
- Visit the app at least once with internet
- Allow site to cache data
- Check Service Worker: Chrome → Settings → Site Settings → MigiTrader
